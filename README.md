# Weather Report

This is a Simple Weather Report Web App using Next.js, the main theme of this project is to work in a collabrative environment with the team members _[Lekhya](https://github.com/Lekhya-18)_, _[Harshitha](https://github.com/SriHarshitha137)_.

## Getting Started

First, run the development server:

```bash
npm install
npm run dev
```

Open _[http://localhost:3000](http://localhost:3000)_ with your browser to see the result.
