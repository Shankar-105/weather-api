# Weather Report

This is a Simple Weather Report Web App using Next.js, the main theme of this project is to work in a collabrative environment with the team members _[Lekhya](https://github.com/Lekhya-18)_, _[Harshitha](https://github.com/SriHarshitha137)_.

## Getting Started

First, run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/app/page.js`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
